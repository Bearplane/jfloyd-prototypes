Q-Audit Lookups — Delete + Reassign workflow

When a Q-Audit Global Settings lookup value (Industry Type, Type of Company, Product Classification, MCC Code) has dependent records, deletion now routes through Delete + Reassign — an atomic migration of dependents to a replacement value before the source is removed. The same modal pattern is shared across all four lookups, with a per-lookup label and dependent-count surface.
