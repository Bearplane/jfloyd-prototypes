Where does + New Case live?

Today, every compliance case is born from items: an analyst selects products / findings, then clicks "Add to case." This exploration flips that — start from a blank case, then target violative findings across the merchant. To make that flow viable, the + New Case CTA must be visible from anywhere a case might be conceived. This page maps the four entry points.
