Per-Company Resolved Scan Config

For a single company, see what Q-ScanAI will scan for, where each setting came from, and override individual elements with one click. Resolution chain: Global default → Industry-pair template(s) (unioned) → Product signals (additive) → Per-company override (always wins). The override layer is the analyst's escape hatch for the rare case where the inherited config doesn't match reality.
