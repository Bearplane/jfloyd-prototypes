Q-Audit · Global Settings — MCC Codes

Adds a new MCC Codes lookup to Q-Audit Global Settings, sibling to Industry Types and Types of Company. Master list of 904 ISO/Mastercard MCCs is imported once and re-imported on demand. Each MCC can be associated with one or more Q-Audit Industry Types so the merchant onboarding form can auto-suggest the right classification when a merchant enters their MCC. Per-row Edit + bulk Assign Industry Type actions; deletion routes through the cross-cutting Delete + Reassign workflow (QP-3766).

Re-import master list — confirmation

Net new modal Idempotent upsert: existing associations preserved
Modal · Confirm re-import

Re-import master MCC list?

Q-Audit will pull the latest iso-18245-2024 master list and reconcile it against the current 904 rows.

What changes
• New MCCs added — appear in the table with no Industry Type associations.
• Title or Sector changed — overwritten in place. Industry Type associations preserved.
• Removed from master — flagged as deprecated but kept (if any company is still using it).
✓ Idempotent · Industry Type associations made by Q-Audit admins are not overwritten.
