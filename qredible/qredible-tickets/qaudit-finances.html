Summary of new features (QP-3547):
1. Date range filter row — From/To date pickers populate via typing or quick-select pills.
2. Quick-select pills: [This Month] [Last Month] [Last 3 Months] [This Year] — clicking auto-fills date inputs and activates filter.
3. Export CSV button — coral/red, right side; exports current filtered (or all) rows.
4. Filtered summary cards — "(filtered)" amber tag appears on all cards + border turns amber when filter is active; values reflect filtered period only.
5. Active filter banner — blue info bar confirms range + row count; includes inline Clear link.
6. Clear filter button — resets date pickers, deselects pills, restores full data + card values.
